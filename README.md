## General info
This Application is intended to provide daily weather information such as
 Min temperature
 Max temperature
 Humidity
 Weather icon
 
 Application retrieve data using darksky API for below cities.
	- Campbell, CA, 
	- Omaha, NE, 
	- Austin, TX,
	- Niseko, Japan,
	- Nara, Japan and
	- Jakarta, Indonesia 

	
## Technologies
Project is created with:
* Java 1.8
* Spring Boot version: 2.2.5.RELEASE
* Mongo DB
* rxjava 1.3.8
